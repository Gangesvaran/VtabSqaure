HTML

HTML stands for Hyper Text Markup Language. It is the standard markup language for creating web pages. It describes the structure of a web page. It consists of a series of a web page. This will help to tell the browser how to display the content.

CSS

CSS stands for Cascading Style Sheets. It is the language we use to style an internet page. It describes how HTML is to be displayed on web page. This saves plenty of labor. It can control the layout of multiple sites all directly.

PHP is a server scripting language. It is a powerful tool for making dynamic and interactive Web pages. It is a widely-used, open deliver scripting language. PHP scripts are accomplished on the server.
